# fuzzy-sugeno-diabetes-detector
Aplikasi web penilai risiko diabetes menggunakan Fuzzy Sugeno Orde-Nol. Sistem mendeteksi dini kesehatan dengan memproses ambiguitas data pasien (Umur, BMI, Gula Darah, Aktivitas Fisik) melalui 32 aturan logika untuk hasilkan skor risiko adaptif. Berjalan murni client-side dengan HTML, CSS, dan JavaScript.
